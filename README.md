# Details
## Summary

Production-ready FlowLet-based 3D brain tumor inpainting framework for **BraTS 2026 Task 5** (healthy tissue reconstruction inside tumor mask).

Faithfully ported and extended from [sisinflab/FlowLet](https://github.com/sisinflab/FlowLet) — all core FlowLet concepts preserved; no replacement with generic UNet or diffusion model.

---

## Architecture: `FlowLet3D_Inpainting`

| Component | Description |
|---|---|
| **DWT3D / IDWT3D** | 3D Haar wavelet DWT/IDWT (ported verbatim from FlowLet transforms.py) |
| **FlowLet3DBlock** | Conditional velocity UNet: ResBlock+FiLM + SpatialAttn+cross-attention grouped into steps (matching FlowLet's TimestepEmbedSequential) |
| **ConditionalFlowMatching3D** | All 4 FlowLet formulations: rectified / CFM / trigonometric / VP-diffusion; Euler ODE sampling |
| **SymmetryAttention3D** | Contralateral hemisphere attention — sagittal flip as anatomical guidance |
| **GlobalContextBranch** | Full-volume voided image encoder |
| **LocalMaskBranch** | Dilated-conv tumor-region encoder |
| **ContextFusion** | Cross-attention fusion of global + local context |
| **Encoder3D / Decoder3D** | Multi-scale 3D U-Net backbone with correct skip connections |

**Dual conditioning**: FiLM modulation (every ResBlock) + cross-attention (cond projected as 1-token sequence context)

## Losses

```
L_total = 1.0 * L_mask + 0.5 * L_global + 0.5 * L_ssim
        + 0.2 * L_flow + 0.1 * L_fft + 0.1 * L_sym + 0.1 * L_edge
```

## Training Features

- AMP · Gradient checkpointing · DDP · EMA (0.9999)
- Cosine LR + linear warm-up · AdamW
- Early stopping on `PSNR_mask` · Best checkpoint saving

## Inference

- `SlidingWindowInferer` (patch 96³, overlap 0.5, Gaussian)
- Test-time augmentation (axis flips)
- Flow-decoder blend at inference
- Preserves original NIfTI affine / spacing / orientation
